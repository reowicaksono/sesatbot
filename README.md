## Installation 📑
* Clone Repository
  ```sh
  git clone https://github.com/reowicaksono/sesatbot
  ```
* Install Modules
  ```sh
  npm install
  ```
* Start
  ```sh
  npm start
  ```
## Start VPS 📑
* Clone Repository
  ```sh
  git clone https://github.com/reowicaksono/sesatbot
  ```
* Install Modules
  ```sh
  npm install
  ```
* Start
  ```sh
  pm2 start
  ```
* Monitoring
  ```sh
  pm2 monit
  ```
